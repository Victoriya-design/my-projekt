# Цели на текущий год

## 3 главные цели

1. Вывести бизнес из убыточного состояния в прибыльный.
2. Повысить узнаваемость бренда.
3. Регулярно вести личные соцсети для развития личного бренда.

## Почему это важно

- Сохранить бизнес и вернуть финансовую устойчивость семьи.
- Обеспечить стабильный доход и рост.

## Цели на ближайшие 90 дней

- Закончить обучение по стилю.
- Запустить базовую автоматизацию управления бизнесом.
- Составить и внедрить план выхода бренда в безубыточность.

## Цели на этот месяц

- Освоить базовую работу в Cursor для ежедневных задач.
- Пройти запланированную часть обучения.
- Подготовить первые ключевые анализы бизнеса с помощью Cursor.

## Привычки, которые поддерживают цели

- Ежедневно: работать в Cursor и в 1С по плану дня.
- Еженедельно: подводить итоги недели и фиксировать прогресс.

## Метрики (как измеряю прогресс)

- Цель 1 (прибыльность): выручка, расходы, чистая прибыль (в рублях) по неделям и за месяц.
- Цель 2 (узнаваемость бренда): охваты, посещаемость, упоминания, входящие запросы.
- Цель 3 (личный бренд): количество публикаций, охваты, подписчики, вовлеченность.
