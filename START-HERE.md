# START HERE

Короткий ежедневный сценарий работы с вашим Second Brain в Cursor.

## 1) Утро (5-10 минут)

1. Откройте `inbox/capture.md` и запишите все мысли/задачи.
2. Откройте `context/goals-current-year.md` и посмотрите 3 главные цели.
3. Сформулируйте 1 главный фокус дня:
   - "Сегодня мой главный результат: ..."

## 2) Рабочий цикл (днем)

Работайте короткими блоками:

1. Берите 1 задачу.
2. Пишите ассистенту: что нужно сделать и какой результат хотите.
3. Делайте по шагам.
4. Фиксируйте результат в нужный файл (`business/`, `notes/`, `social-media/` и т.д.).

Если не знаете, с чего начать, пишите:

"Я новичок, дай 3 простых шага для этой задачи."

## 3) Вечер (5-10 минут)

1. Обновите `context/memory.md`:
   - что сработало;
   - что было сложно;
   - как лучше работать завтра.
2. В `inbox/capture.md` оставьте список задач на завтра.
3. Коротко подведите итог дня (2-3 строки).

## 4) Когда сохранять в GitHub

Сохраняйте после любого полезного блока изменений (например, 1-2 завершенные задачи).

Команды:

```powershell
cd C:\repo
git status
git add .
git commit -m "Коротко: что сделано"
git push
```

## 5) Правило безопасности

Перед любыми внешними действиями (публикации, отправка сообщений, интеграции, удаление) ассистент должен спросить подтверждение.

## 6) Готовые фразы для старта

- "Разложи `inbox/capture.md` по папкам."
- "Сделай план на сегодня по моим целям."
- "Помоги оформить задачу в понятный чек-лист."
- "Проверь, что я не забыла перед публикацией."

---

Мини-цель: каждый день хотя бы 1 завершенная задача + 1 короткая фиксация результата в файлах.
